# Tanglaw: Philippine Journal of Student and Teacher Research

This package is a launch-ready static website for a GitHub Pages journal site.

## Suggested repository name

`tanglaw-journal`

If you create a new repository under the `padadanhs` account and upload these files to the repository root, the site URL will typically be:

`https://padadanhs.github.io/tanglaw-journal/`

## Included pages

- Home
- About
- Aims and Scope
- Editorial Board
- For Authors
- Submit
- Articles
- Archive
- News
- Ethics Policy
- Peer Review Policy
- Privacy Statement
- Open Access Policy
- Contact
- Sample article pages
- Sample issue page
- 404 page

## Before public launch

1. Replace the placeholder email `samueljohn.parreno@deped.gov.ph` everywhere with your real journal email.
2. Update the editorial board names on `/editorial-board/`.
3. Replace sample articles and news items with your real content.
4. Adjust the repository name in your own GitHub account if desired.

## Folder-based clean URLs

Each page is placed in its own folder with `index.html`, so URLs remain clean, such as:

- `/about/`
- `/aims-and-scope/`
- `/for-authors/`
- `/submit/`
- `/archive/`
- `/articles/`

## GitHub Pages deployment

1. Create a new repository under the `padadanhs` account.
2. Upload all extracted files to the repository root.
3. Go to **Settings → Pages**.
4. Set **Deploy from branch** to `main` and folder to `/root`.
5. Save and wait for GitHub Pages to publish.
